## Basic code to create a horizontal menu. 
```css
#menu ul{
    margin: 0;
    padding: 0;
    list-style: none; 
}
#menu li{
    display: inline;
    position: relative; left: 300px;
    padding-left: 60px;
}
```
Changing the color and decoration of a link 
```css
a:link{
    color:darkcyan;
    text-decoration: none;
}
a:visited{
    color:plum;
    text-decoration: none;
}
a:hover{
    color: plum;
    text-decoration: underline;
}
```
